# OrgCheck

**Namespace Prefix:** `orgcheck`

## Description

OrgCheck is a Salesforce application and `sf` CLI plugin that quickly analyzes your org and surfaces technical debt. It
was created by Salesforce Labs as an open-source alternative to the sunsetting Salesforce Optimizer.

## Problems It Solves

- Identifies bad practices and technical debt across your Salesforce org
- Provides continuous org health monitoring without adding custom objects or external connections
- Enables both interactive (UI) and automated (CLI) org analysis workflows
- Keeps all data local — nothing leaves your browser or terminal

## Key Features

- Analyze Apex classes, hardcoded URLs, and a global org health view
- Export results as CSV, JSON, or XLSX
- No custom objects created; no external connections required
- Supports multi-org automation scenarios via `sf` CLI plugin commands (`sf check apex-classes`,
  `sf check hardcoded-urls`, `sf check global-view`)
- Available on AppExchange and as an open-source project on GitHub
