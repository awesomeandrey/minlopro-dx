# User Access and Permissions Assistant

**Namespace Prefix:** `Profile2PermSet`

> Formerly known as **Profile2PermSet**

## Description

User Access and Permissions Assistant (UAPA) is a free Salesforce AppExchange app that gives admins the ability to
analyze, report, and manage permissions and permission assignments across their org. It is the go-to tool for orgs
migrating from profiles to permission sets.

## Problems It Solves

- Difficulty auditing which users hold which permissions across profiles, permission sets, and permission set groups
- Manual, error-prone process of converting profiles into permission sets
- Lack of a unified UI to manage and compare permission assignments at scale
- No built-in reporting on effective user access and permission set group composition

## Key Features

- Convert profiles into editable permission sets directly from the UI
- Analyze and report on permissions and access across the entire org
- Manage permission set groups and the permissions they contain
- Assign permission sets to users in bulk
- Requires the Tooling API to be enabled in the org
